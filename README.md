# Pre-Interview Test: Merge Function ## Setup & Run Instructions 
### Prerequisites Ensure you have the following installed:
* **Node.js** (v18+ recommended)
* **npm** (comes with Node.js)

npm (comes with Node.js)

Verify your installation:
```bash

node -v
npm -v
```

Installation

Clone the repository and install dependencies (TypeScript, Jest, ts-jest):
```bash


git clone https://github.com/rennik98/Primo-Interview-Test.git
cd Primo-Interview-Test
npm install

```

Project Structure
```bash
src/merge.ts        → Merge function implementation
tests/merge.test.ts → Unit tests
jest.config.js      → Jest configuration
tsconfig.json       → TypeScript configuration
```
Run Unit Tests

Execute the following command to run all unit tests:
```bash

npm test

```

Expected Result

All tests pass successfully

The merge function returns a single ascending sorted array

No built-in sort functions are used in the implementation
